# Parede de certificados

O projeto consiste em um mural mostrando todos os meus certificados adiquiridos durante a vida começando do meu certificado de conclusão do ensino medio até meu diploma da faculdade, o que a primeira vista parece pouco porem são varios feitos tanto com curso online quanto aulas extras presenciais e badges como conteudo extra (aside)

esses certificados são minhas provas de conhecimento das areas que estou estudando e esse é um projeto continuo, ou seja conforme adiquiro novos certificados atualizo a pagina
